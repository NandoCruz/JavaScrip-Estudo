Estudando JavaScript

Criando uma página usando os comandos: alert, confirm e prompt.


A primeira tela que apareceu para você foi utilizado o comando "window.alert" onde apareceu na tela a seguinte mensagem "Você está estudando JavaScript".

window.alert('Você está estudando JavaScript')

A segunda tela que apareceu para você foi utilizado o comando "window.confirm" onde apareceu na tela a seguinte mensagem "Você está gostando curso?" e tem dois botões de confirmação.

window.confirm('Você está gostando do curso?')

A terceira tela que apareceu paravocê foiutilizado o comando "window.prompt" onde apareceu uma tela para você digitar o seu nome com a pergunta "Qual o seu nome?".

window.prompt('Qual o seu nome?')
